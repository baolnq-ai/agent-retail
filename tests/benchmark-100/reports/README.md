# Report benchmark

Thư mục này lưu kết quả chạy benchmark 100 câu.

Mỗi lần chạy nên có:

- File JSON đầy đủ để đọc lại response, trace, block và issue từng case.
- File Markdown tóm tắt để reviewer đọc nhanh pass/warn/fail, lỗi flow/tool/prompt và khuyến nghị fix.
- Checkpoint đang chạy nếu benchmark bị dừng giữa chừng.
