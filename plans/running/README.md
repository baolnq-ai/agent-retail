# Running Plans

- Created: 2026-05-26
- Scope: các plan đang chạy, chưa đạt close criteria hoặc chưa được đóng log/doc cuối.

## Cách Dùng

- Mỗi task nhiều bước tạo một file `plan-{task-name}-YYYYMMDD-v1.md` trong thư mục này.
- Khi task hoàn thành, chuyển plan ra thư mục phù hợp dưới `plans/` hoặc root `plans/` theo convention đang dùng.
- Plan đang chạy phải có `Related log` và `Related doc` nếu task đủ quan trọng để cần truy vết sau này.
