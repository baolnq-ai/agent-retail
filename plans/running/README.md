## Cách Dùng

- Mỗi task nhiều bước tạo một file `plan-{task-name}-YYYYMMDD-v1.md` trong thư mục này.
- Khi task hoàn thành, chuyển plan ra thư mục phù hợp dưới `plans/` hoặc root `plans/` theo convention hiện tại.
- Plan đang chạy phải có `Related log` và `Related doc` nếu task đủ quan trọng để cần truy vết sau này.
