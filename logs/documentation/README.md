# Documentation Logs

- Created: 2026-05-26
- Scope: log cho các task tạo, chỉnh, audit hoặc chuẩn hóa tài liệu, plan, evidence và format skill.

## Cách Dùng

- Mỗi log ghi ngắn gọn mục tiêu, thời gian, phase chính, file đã sửa, verify đã chạy và rủi ro còn lại.
- Log phải link về plan và doc nếu task có plan/doc riêng.
- Không copy output dài, không ghi secret, token, credential hoặc dữ liệu nhạy cảm.
